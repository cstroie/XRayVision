XRayVision is an open-source system for real-time DICOM ingestion, processing, and interactive review of radiology studies using AI assistance (OpenAI).

  • 🧠 Integrates with OpenAI (primary & fallback) for image interpretation
  • 📦 Supports storescp-based DICOM reception
  • 📊 Real-time WebSocket dashboard with live queue and result updates
  • 🗂 SQLite-based exams with pagination, filtering, and flagging
  • 📈 Statistics with anatomy breakdown and health monitoring
  • 🌐 Fully async and responsive, built with aiohttp + PicoCSS
